# front_end

## 环境安装

* 安装 nodejs

  ```
  apt-get update
  apt-get install curl
  curl -SLO https://deb.nodesource.com/nsolid_setup_deb.sh
  chmod 500 nsolid_setup_deb.sh
  ./nsolid_setup_deb.sh 21
  apt-get install nodejs -y
  ```

* 安装Vue

  ```
  npm install -g @vue/cli 
  ```

npm install
